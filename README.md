# Personal Telegram AI Assistant — Фаза 1

Личный AI-ассистент в Telegram. Стоимость: **$0–5/мес**.

---

## Стек

- **LLM**: Groq (Llama 3.3 70B) — бесплатно
- **Голос**: Whisper Large v3 через Groq — бесплатно
- **Bot**: aiogram 3 + Telegram Bot API
- **Память**: SQLite + FTS5 поиск
- **Хостинг**: Railway (бесплатно → $5/мес)

---

## Быстрый старт

### 1. Получить токены

**Telegram бот:**
1. Открыть @BotFather в Telegram
2. `/newbot` → придумать имя → получить токен

**Узнать свой Telegram ID:**
1. Написать @userinfobot
2. Скопировать `Id:` число

**Groq API (бесплатно):**
1. Зарегистрироваться на [console.groq.com](https://console.groq.com)
2. API Keys → Create → скопировать

---

### 2. Локальный запуск (для теста)

```bash
git clone <repo>
cd assistant

# Установить зависимости
pip install -r requirements.txt

# Настроить переменные
cp .env.example .env
# Открыть .env и заполнить свои значения

# Запустить
python main.py
```

---

### 3. Деплой на Railway (рекомендуется)

1. Зарегистрироваться на [railway.app](https://railway.app)
2. New Project → Deploy from GitHub repo
3. Загрузить код в GitHub репозиторий
4. В Railway: Variables → добавить все переменные из `.env.example`
5. Deploy → готово

Бот работает 24/7, деплой занимает 2 минуты.

## Команды бота

| Команда | Действие |
|---------|----------|
| `/start` | Приветствие и список команд |
| `/setup` | Настроить профиль (имя, цели, стиль) |
| `/tasks` | Список активных задач |
| `/done [id]` | Отметить задачу выполненной |
| `/reminders` | Список напоминаний |
| `/search [запрос]` | Поиск по заметкам (FTS5) |
| `/profile` | Показать профиль и факты |
| `/reset` | Очистить историю диалога |

## Что умеет

### Автоматически (через tool calling)
- 📋 Создаёт задачи с приоритетом и дедлайном
- 📝 Сохраняет заметки и идеи с тегами
- ⏰ Ставит напоминания (проверка каждую минуту)
- 🔍 Ищет по заметкам через FTS5
- 📌 Запоминает важные факты о тебе навсегда

### Память
- Последние 20 сообщений → в контекст LLM
- Профиль (имя, цели, стиль) → всегда в системном промпте
- Закреплённые факты → всегда в системном промпте
- Все заметки → FTS5 поиск по запросу

### Голос
- Отправляй голосовые — Whisper распознаёт на русском
- Обрабатываются так же как текст

## Примеры использования

```
«Создай задачу: написать отчёт, высокий приоритет, до пятницы»
→ Создаёт задачу автоматически

«Напомни мне завтра в 10:00 позвонить Ивану»
→ Ставит напоминание

«Запиши идею: сделать интеграцию с календарём»
→ Сохраняет заметку

«Найди заметки про маркетинг»
→ FTS5 поиск, показывает результаты

«Меня зовут Макс, моя цель — запустить SaaS»
→ Сохраняет в профиль
```

## Архитектура

```
Telegram
   ↓
aiogram 3 (handlers)
   ↓
SQLite (history + profile + pinned)
   ↓
Groq API (Llama 3.3 70B)
   ↓ tool calling
Tools: tasks / notes / reminders / search / pin
   ↓
Ответ → Telegram
```
